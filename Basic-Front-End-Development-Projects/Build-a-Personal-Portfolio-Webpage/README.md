# 建立个人档案页面（Build a Personal Portfolio Webpage）任务

## 任务要求
* 一个Web页面
* 可以随意定义风格,不仅仅是Bootstrap。
* 以通过滚动来浏览这个页面的所有内容。
* 可以通过点击不同的按钮来跳到不同的社交媒体页。
* 你可以看到不同项目的缩略图。

## 实现说明
图片利用tumblr.com上传加载。

## 项目结构
* index.html - 入口和所有内容的展示
